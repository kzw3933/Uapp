#### Android闪屏页、启动页实现

```
启动页
    - 作用: 缓解用户焦虑、运行业务的处理(网络检测...)
    - 界面组成: Logo、背景图、版本号等元素
闪屏页
    - 作用: 营销推广(停留三秒，跳过或者进入详情页)
    - 界面组成: 广告、Logo、跳过(倒计时按钮)
引导页
    - 作用: 引导用户了解产品特色、关键功能(可以直接写在App里，一般是3-5屏左右、"立即体验")
    - 使用场景: 用户首次安装或者版本更新后、第一次打开App的时候出现、之后不再出现
```
